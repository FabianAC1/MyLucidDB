Provides a language for representing row-expressions.

Life-cycle

A {@link net.sf.saffron.sql2rel.SqlToRelConverter} converts a SQL parse tree consisting of {@link net.sf.saffron.sql.SqlNode} objects into a relational expression ({@link net.sf.saffron.rel.SaffronRel}). Several kinds of nodes in this tree have row expressions ({@link net.sf.saffron.rex.RexNode}).

After the relational expression has been optimized, a {@link net.sf.saffron.oj.rel.JavaRelImplementor} converts it into to a plan. If the plan is a Java parse tree, row-expressions are translated into equivalent Java expressions.

Expressions

Every row-expression has a type. (Compare with {@link net.sf.saffron.sql.SqlNode}, which is created before validation, and therefore types may not be available.)

Every node in the parse tree is a {@link net.sf.saffron.rex.RexNode}. Sub-types are:

  • {@link net.sf.saffron.rex.RexLiteral} represents a boolean, numeric, string, or date constant, or the value NULL.
  • {@link net.sf.saffron.rex.RexVariable} represents a leaf of the tree. It has sub-types:
    • {@link net.sf.saffron.rex.RexCorrelVariable} is a correlating variable for nested-loop joins
    • {@link net.sf.saffron.rex.RexInputRef} refers to a field of an input relational expression
    • {@link net.sf.saffron.oj.xlat.AggInternalTranslator.RexGroupVariable} and {@link net.sf.saffron.oj.xlat.AggInternalTranslator.RexAggVariable} are created temporarily while an aggregate expression is being translated
  • {@link net.sf.saffron.rex.RexCall} is a call to an operator or function. By means of special operators, we can use this construct to represent virtually every non-leaf node in the tree.
  • {@link net.sf.saffron.rex.RexRangeRef} refers to a collection of contiguous fields from an input relational expression. It usually exists only during translation.

Expressions are generally created using a {@link net.sf.saffron.rex.RexBuilder} factory. If Java expressions are to be converted, you can use a derived class, {@link net.sf.saffron.oj.util.JavaRexBuilder}.

{@link net.sf.saffron.oj.xlat.InternalTranslator} translates a Java row-expression ({@link openjava.ptree.ParseTree}) into a row-expression, all within the context of a particular {@link net.sf.saffron.oj.xlat.QueryInfo} and input relational expressions ({@link net.sf.saffron.rel.SaffronRel}).
