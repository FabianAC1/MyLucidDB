Related packages and classes

  • Package net.sf.saffron.sql is an object model for SQL expressions
  • Package openjava.ptree defines an object model for Java expressions.
  • Class {@link net.sf.saffron.oj.xlat.SqlToOpenjavaConverter} converts SQL expressions to Java expressions (including extensions such as {@link openjava.ptree.QueryExpression}).
  • Class {@link net.sf.saffron.oj.xlat.OJQueryExpander} converts a Java expression to a {@link net.sf.saffron.rel.SaffronRel relational expression}.
  • Package saffron.opt provides an optimizer.
