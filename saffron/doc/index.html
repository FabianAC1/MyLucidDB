Saffron

Saffron is an extension to Java to incorporate SQL-like relational expressions. It features a pre-processor, a query optimizer, and a runtime library.

Read the overview, installation instructions, and javadoc.

The project is hosted at SourceForge.

The source code (including this page) is hosted in John Sichi's perforce server. Thanks to John, and to Perforce for letting us use their server gratis.
